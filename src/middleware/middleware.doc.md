# Middleware Documentation
## JWT Decoder
The decoder is a function which accepts a token (as Buffer or string) and
returns the payload or the sections of the token.  
The following example is from the `fast-jwt` documentation. It help understand 
how to use JWT.
```ts
// Example from 
const { createDecoder } = require('fast-jwt')
const token = 'token-generated-by-signer'

// Standard decoder
const decode = createDecoder()
const payload = decode(token)
// => { a: 1, b: 2, c: 3, iat: 1579521212 }
```
There's also a complete decoder. Follow this [link](https://www.npmjs.com/package/fast-jwt#examples) to read more.
## JWT Signer
Same as before, with the decoder:
```ts
const { createSigner } = require('fast-jwt')

// Sync style
const signSync - createSigner({ key: 'secret' })
cosnt token = signSync({ a: 1, b: 2, c: 3 })
// => token-generated-by-signer
```
Use the token generated by the signer to decode the incoming payload from the 
request. Visit [this link](https://www.npmjs.com/package/fast-jwt#example) for 
more examples.
